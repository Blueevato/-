# 多用户通讯系统 - Go 语言实现

一个基于 Go 语言开发的多用户实时通讯系统，支持用户注册、登录、在线用户显示和消息发送功能。

## 🚀 功能特性

- ✅ 用户注册与登录
- ✅ 显示所有在线用户
- ✅ 群发消息
- ✅ 消息记录查看
- 🔄 私聊功能//TODO
- 💾 Redis 数据存储
- 🏗️ 客户端-服务端架构

## 📋 前置要求

- Go 1.16 或更高版本
- Redis 服务器
- Git

## 🛠️ 部署教程

### 1. 启动 Redis 服务

```bash
# 在 Linux/macOS 上
redis-server

# 在 Windows 上
# 启动 Redis 服务，或使用 Redis Windows 版本

2. 部署并启动服务
服务端部署
bash
# 进入服务端目录
cd server

# 编译服务端
go build -o chat-server ./main

# 启动服务端
./chat-server
# 或者直接运行
go run ./main

客户端部署
bash
# 进入客户端目录
cd client/client_main

# 编译客户端
go build -o chat-client .

# 启动客户端
./chat-client
# 或者直接运行
go run .
